Installing Ubuntu on Windows 10

  • Thu 25 August 2022
  • Linux

Intro:

  • Sometimes we may want to run a Linux environment on windows to run Linux commands, we can do this by creating virtual machine on windows system but it would take more resources(CPU, Memory and storage)
  • But there is an another option called WSL aka Windows Subsystem for Linux which would require less resources than a full Virtual Machine.
  • WSL also allows you to run Linux command-line tools and apps alongside your Windows command-line also you can access your Windows files from within Linux.

how to install Linux system on Windows 10 using WSL:

  1. Search for "Turn Windows features On or Off" in windows search

  1. It will open a window as below, scroll down and select 'Windows subsystems for Linux' and click on OK.

  1. Once you click on OK, it will apply required changes and asks for your system restart.

  1. After the restart, we can install our preferred Linux Distributions using "Microsoft Store" in your system, search for 'Ubuntu' or 'Linux'.

  1. Here, I am trying to install Ubuntu(Linux distribution)

  2. Once installed, open the app, it will take few moments to finish the installation and will launch a Ubuntu Terminal.

  3. Inside the terminal, you will be asked to enter Unix username and password, which is different from your windows credentials, so you can give new credentials here.

  4. Finally, we can run Linux commands in the Ubuntu terminal

  5. ls -al, will list the files and directories in the current directory

How to navigate through Windows drives,

  • enter below commands sequentially

    1. ls /mnt/
    2. cd /mnt/c
    3. ls -al, cd Users/your_desktop_path/

Simple Linux Commands :

  • touch notes.txt -- to create notes.txt text file
  • nano notes.txt -- opens the text file using default and simple terminal-based text editor
  • cat notes.txt --displays text in the file
  • pwd -- path of the current working directory (folder)
  • mkdir Music -- creates a new 'Music' folder

you can find and learn more about Linux command line here

Also, you can open bash terminal in windows command prompt window by entering, bash -- and run Linux commands
